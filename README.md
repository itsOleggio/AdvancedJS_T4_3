# Задание 4.3 "Коллекция Map"
![CI](https://github.com/itsOleggio/AdvancedJS_T4_3/actions/workflows/ci.yml/badge.svg)

## Описание 

Изменение состояний приложения и настроек пользователя

## Тесты

Используем библиотеку Jest.
Тестом покрыта основная функция программы 
- `set(name, value)` - устанавливает значение в Map
- `get settings()` - возвращает Map


Запуск тестов: `npx test`. 

## Код программы

Программа находится в папке `src`.
Тесты ёт в папке `/src/ts/__tests__/`.

## Запуск программы

Сборка программы: `npm build`
Запуск программы: `npm start`.
